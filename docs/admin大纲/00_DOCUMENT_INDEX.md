## 📝 文档维护原则

- **任何架构级变更都必须先更新文档,再写代码**
- 文档版本采用 vX.Y 标记,大版本更新时旧版本归档保留
- 所有新增模块必须创建对应 SPEC,并在本目录登记
- AI Agent 修改代码前,**强制阅读对应模块的 SPEC**
- 文档使用 Markdown 编写,纯文本可读
