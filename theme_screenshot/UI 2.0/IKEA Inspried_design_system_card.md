# Design System: Sjöbo Marketplace (IKEA Inspired)

## 1. Creative North Star: "Democratic Design for Campus"
Built on the principles of Swedish functionalism: clean, accessible, and high-quality design for everyone. High-contrast signaling and clear structure.

## 2. Color Palette
- **IKEA Blue:** `#0058AB` — Primary brand color for CTAs and headers.
- **IKEA Yellow:** `#FDD816` — Secondary accent for attention and highlights.
- **Surface:** `#FFFFFF` — Main background for clarity.
- **Neutral Gray:** `#F5F5F5` — Subtle sectioning and card backgrounds.
- **On Surface:** `#1A1C1C` — Primary text color for high readability.
- **On Surface Variant:** `#767676` — Secondary text and labels.

## 3. Typography (Plus Jakarta Sans)
Clean, professional, and functional.
- **Display (Hero):** 36px / 700 (Bold) / #1A1C1C
- **Headline (Page Titles):** 24px / 700 (Bold) / #1A1C1C
- **Title (Card Titles):** 18px / 600 (Semi-Bold) / #1A1C1C
- **Body (Standard):** 14px / 400 (Regular) / #1A1C1C
- **Label (Utility):** 12px / 600 (Semi-Bold) / Uppercase / #767676

## 4. Layout & Components
- **Corner Radius:** 4px — Structured and functional.
- **Margins:** 24px screen margins / 16px card gutters.
- **Primary Button:** Solid IKEA Blue background with White text.
- **Secondary Button:** Solid IKEA Yellow background with IKEA Blue text.
- **Navigation:** Clean, icon-based top and bottom bars with no-line architecture.
